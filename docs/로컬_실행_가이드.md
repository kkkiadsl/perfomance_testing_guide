# Docker 없이 로컬에서 실습하기

Docker를 사용할 수 없는 환경에서 터미널 명령어만으로 실습 환경을 구성하는 가이드입니다.

---

## 사전 준비

### 1. Python 패키지 설치 (대상 서버 + Locust)

프로젝트 루트 디렉토리에서 아래 명령어를 실행합니다.

```bash
pip install -r requirements.txt
```

`requirements.txt`에 포함된 패키지:

| 패키지 | 용도 |
|---|---|
| `fastapi` | 대상 서버 웹 프레임워크 |
| `uvicorn[standard]` | FastAPI ASGI 서버 |
| `pydantic` | 데이터 유효성 검사 |
| `locust` | 성능 테스트 도구 |

> **Python 버전**: 3.9 이상 권장

---

### 2. k6 설치 (macOS — Homebrew)

```bash
brew install k6
```

설치 확인:

```bash
k6 version
```

---

## 실행 순서

실습은 **터미널 3개**를 열어 각각 실행합니다.

```
터미널 1 → 대상 서버 (target-server)
터미널 2 → Locust 또는 k6
```

---

## 터미널 1 — 대상 서버 실행

```bash
cd performance_test/target-server

uvicorn app:app --host 0.0.0.0 --port 8080 --workers 1
```

서버 시작 확인:
- 대시보드: http://localhost:8080
- 헬스체크: http://localhost:8080/health

> `--workers 1` 옵션은 필수입니다. 인메모리 메트릭 수집기가 단일 프로세스에서만 정상 동작합니다.

---

## 터미널 2 — Locust 실행

```bash
cd performance_test/locust/scripts

locust -f locustfile.py --host http://localhost:8080
```

Locust 웹 UI 접속: http://localhost:8089

웹 UI에서 설정:
- **Number of users**: 동시 사용자 수 (예: 50)
- **Ramp up**: 초당 증가 사용자 수 (예: 5)
- **Host**: `http://localhost:8080` (자동 입력됨)

### Locust 헤드리스 모드 (웹 UI 없이 실행)

```bash
locust -f locustfile.py \
  --host http://localhost:8080 \
  --headless \
  --users 50 \
  --spawn-rate 5 \
  --run-time 3m
```

| 옵션 | 설명 |
|---|---|
| `--headless` | 웹 UI 없이 터미널에서 바로 실행 |
| `--users` | 최대 동시 사용자 수 |
| `--spawn-rate` | 초당 사용자 증가 속도 |
| `--run-time` | 테스트 실행 시간 (예: `3m`, `120s`) |

---

## 터미널 2 — k6 실행

```bash
cd performance_test/k6/scripts

k6 run example.js
```

### 환경 변수로 대상 URL 변경

```bash
TARGET_URL=http://localhost:8080 k6 run example.js
```

### 실행 옵션 오버라이드 (스크립트 설정 무시하고 직접 지정)

```bash
# 가상 사용자 10명, 30초 실행
k6 run --vus 10 --duration 30s example.js

# 단계적 부하 증가 (Ramp-up)
k6 run --stage 30s:10,1m:50,30s:0 example.js
```

| 옵션 | 설명 |
|---|---|
| `--vus` | 동시 가상 사용자(Virtual Users) 수 |
| `--duration` | 테스트 실행 시간 |
| `--stage` | `시간:사용자수` 형식으로 단계 지정 |

### k6 결과를 JSON 파일로 저장

```bash
k6 run --out json=result.json example.js
```

## 빠른 참조

```bash
# 대상 서버
uvicorn app:app --host 0.0.0.0 --port 8080 --workers 1

# Locust (웹 UI)
locust -f locustfile.py --host http://localhost:8080

# Locust (헤드리스)
locust -f locustfile.py --host http://localhost:8080 --headless --users 50 --spawn-rate 5 --run-time 3m

# k6 기본 실행
k6 run example.js

# k6 사용자/시간 직접 지정
k6 run --vus 10 --duration 30s example.js
```

## 대상 서버 엔드포인트

| 메서드 | 경로 | 설명 |
|---|---|---|
| GET | `/` | 실시간 모니터링 대시보드 |
| GET | `/health` | 헬스체크 |
| GET | `/api/items` | 상품 목록 (`?page=1&size=20`) |
| GET | `/api/items/{id}` | 상품 상세 |
| POST | `/api/items` | 상품 생성 |
| GET | `/api/slow` | 지연 시뮬레이션 (`?delay=초`) |
| GET | `/api/error` | 에러율 시뮬레이션 (`?rate=0~1`) |
