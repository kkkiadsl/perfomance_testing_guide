# 서버 성능 테스트 실습을 위한 환경 준비 가이드

**QA 엔지니어를 위한 서버 성능 테스트 입문**

---

## 개요

강의 당일 원활한 실습을 위해 **강의 전날까지** 아래 2가지를 설치해 주세요.
라이브러리 설치 및 실습 진행은 강의 시간에 함께 합니다.

**준비 항목**:
1. VS Code 설치
2. Python 설치

---

## 1단계 — VS Code 설치

코드를 편집하고 터미널을 실행하는 도구입니다.

**[code.visualstudio.com](https://code.visualstudio.com/)** 접속 → OS에 맞는 버전 다운로드 후 설치

설치 후 VS Code를 실행하여 아래 화면이 뜨면 완료입니다.

```
┌─────────────────────────────────┐
│  Visual Studio Code             │
│                                 │
│  Get Started                    │
│  Open Folder...                 │
└─────────────────────────────────┘
```

### 확장 프로그램 설치

VS Code 좌측 Extensions 아이콘 클릭 → **Python** (Microsoft) 검색 → Install

---

## 2단계 — Python 설치

실습 도구(Locust)와 대상 서버를 실행하는 데 필요합니다.

### macOS

```bash
brew install python
```

> Homebrew가 없다면 먼저 설치:
> `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

### Windows

**[python.org/downloads](https://www.python.org/downloads/)** 접속 → 최신 버전 다운로드 후 설치

> 설치 화면에서 **"Add Python to PATH"** 체크박스를 반드시 선택하세요.

### 설치 확인

VS Code에서 터미널을 열고 (`Ctrl + `` ` ``) 아래 명령어를 실행합니다.

```bash
python3 --version
```

**정상 출력 예시**:
```
Python 3.11.5
```

버전이 **3.9 이상**이면 준비 완료입니다.

## 참고 — 터미널과 VS Code 간략 안내

### 터미널

명령어를 입력해서 컴퓨터를 제어하는 텍스트 창입니다.
서버 실행, 패키지 설치 등을 모두 터미널에서 진행합니다.

**VS Code에서 터미널 열기**: `` Ctrl + ` `` (백틱, 숫자 1 왼쪽 키)

꼭 알아야 할 명령어:

| 명령어 | 설명 |
|---|---|
| `cd 폴더명` | 해당 폴더로 이동 |
| `cd ..` | 상위 폴더로 이동 |
| `Ctrl + C` | 실행 중인 명령어 종료 |

### VS Code

실습에서 사용하는 기능:

| 기능 | 방법 |
|---|---|
| 폴더 열기 | `File` → `Open Folder` |
| 터미널 열기 | `` Ctrl + ` `` |
| 터미널 추가 | 터미널 패널 우측 `+` 버튼 |
| 파일 저장 | `Ctrl + S` / `Cmd + S` |

> 나머지 사용법은 강의 시간에 함께 안내합니다.
